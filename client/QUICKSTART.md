# 🚀 Quick Start Guide

## Get Your Resume Builder Running in 2 Minutes!

### Step 1: Open Terminal/Command Prompt
Navigate to the client folder:
```bash
cd "C:\Users\RITHIK S\RESUME-BUILDER\client"
```

### Step 2: Start the Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Once the server starts, you'll see output like:
```
  VITE v7.2.4  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

Open your browser and go to: **http://localhost:5173**

---

## 🎯 Using the Resume Builder

### 1️⃣ **Dashboard**
- Click **"Create Resume"** to start from scratch
- Or **"Upload Existing"** to import a PDF resume

### 2️⃣ **Edit Your Resume**
Navigate through the tabs:
- **👤 Personal Info** - Add your contact details
- **📝 Summary** - Write your professional summary (use AI Suggestions!)
- **💼 Experience** - Add your work history
- **🎓 Education** - List your degrees
- **⚡ Skills** - Add skills (press Enter to add quickly)
- **🚀 Projects** - Showcase your projects
- **🎨 Theme** - Choose template and colors

### 3️⃣ **Interactive Features**
- ✨ Click **"AI Suggestions"** for smart content ideas
- 💾 Click **"Save"** to save your changes
- 👁️ Click **"Preview"** to see your resume
- 🎨 Choose from 4 templates and 7 color schemes

### 4️⃣ **Preview & Export**
- Click **"Edit"** to go back to editing
- Click **"Share"** to copy a shareable link
- Click **"Download PDF"** (coming soon!)

## 💡 Pro Tips

1. **Use AI Suggestions** - Click the sparkle button for professionally written content
2. **Save Often** - Click the Save button after major changes
3. **Try Different Templates** - Switch templates in the Theme tab to see what looks best
4. **Empty States** - Don't worry if sections are empty, the app will guide you!
5. **Character Count** - Watch the character count in the Summary section

## 🆘 Troubleshooting

### Port Already in Use?
```bash
npm run dev -- --port 3000
```

### Styling Not Loading?
Make sure Tailwind is installed:
```bash
npm install
```

### Page Not Loading?
Check if the dev server is running. You should see:
```
VITE v7.2.4  ready
```
